# Work Resources

Static GitHub Pages directory for work resources and Beaumont Division assignment schedules.

## Contents

- `index.html` — homepage.
- `resources/schedules/beaumont-assignment-schedule.html` — printable/shareable schedule page.
- `data/jobs.csv` — job schedule data.
- `assets/css/style.css` — styling.
- `assets/js/main.js` — renders the schedule tables from CSV.
